📒 İşlem Günlüğü

ⓘ Rapor / işlem geçmişi nasıl yüklenir?

İşlemlerini elle girmek yerine broker'ından aldığın raporu yükleyerek otomatik aktarabilirsin: profil seç (yoksa “+ Yeni profil” aç) → 📥 Rapor Yükle → dosyanı seç. Desteklenen formatlar: HTML, CSV, PDF.

🟦 MT5 (MetaTrader) kullananlar

  1. MetaTrader 5'i aç.
  2. Alt taraftaki Araç Kutusu → Geçmiş (History) sekmesine gel.
  3. Sağ tık → Rapor (Report) → HTML olarak kaydet.
  4. Oluşan HTML dosyasını siteye yükle.

🟨 Tickmill

  1. Tickmill panelinde İşlem / Emir Geçmişi ekranına gir.
  2. “Güncelleme Saati” sütunu olan (tarihli) CSV'yi dışa aktar.
  3. Bu CSV'yi siteye yükle — işlemler gün gün takvime dağılır.

🟩 TradingView üzerinden işlem yapanlar

  1. TradingView'e bağladığın broker hesabının emir geçmişini indir.
  2. İndirdiğin dosyayı (tercihen tarihli CSV) siteye yükle.

🟧 Fon firması (prop) hesabı

  1. Fon firması panelinde / platformunda “Generate Statement” (statement oluştur) seçeneğine tıkla.
  2. Oluşan statement dosyasını (PDF/HTML) indir.
  3. Bu dosyayı buraya yükle.

🔒 Yüklediğin veriler yalnızca senin cihazında saklanır, kimseyle paylaşılmaz.
